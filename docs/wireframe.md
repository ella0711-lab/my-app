# 마이링크 (MyLink) UI 구조 및 와이어프레임 설계

본 문서는 마이링크 서비스의 UI 컴포넌트 구조, 페이지 흐름, 그리고 주요 화면에 대한 와이어프레임을 다이어그램과 ASCII 아트를 사용하여 정의합니다.

---

## 1. UI 구조 다이어그램 (Mermaid)

### 1.1. 페이지 간 화면 흐름도 (Screen Flow)
방문자와 서비스 소유자(크리에이터) 각각의 진입 및 화면 이동 경로를 보여줍니다.

```mermaid
graph TD
    %% 방문자 흐름
    Visitor[방문자] -->|개인 링크 URL 접속| BioPage[마이링크 페이지]
    BioPage -->|소셜 아이콘 클릭| SocialMedia[외부 SNS 채널]
    BioPage -->|링크 카드 클릭| ExternalLink[외부 콘텐츠/상품 페이지]

    %% 소유자 흐름
    Owner[소유자] -->|접속| Login[로그인 페이지]
    Login -->|계정 인증| AdminDashboard[관리자 대시보드]
    AdminDashboard -->|프로필 정보 변경| PreviewUpdate[실시간 프리뷰 업데이트]
    AdminDashboard -->|링크 추가/수정/삭제| PreviewUpdate
    AdminDashboard -->|테마 및 스타일 변경| PreviewUpdate
    AdminDashboard -->|통계 확인| Analytics[통계 대시보드]
```

### 1.2. 컴포넌트 계층 구조 (Component Hierarchy)
화면에 렌더링되는 구성 요소들의 구조적 계층 구조입니다.

```mermaid
graph TD
    App[마이링크 서비스] --> BioPage[방문자 개인 페이지]
    App --> AdminPage[관리자 대시보드]

    %% 방문자 페이지 컴포넌트
    BioPage --> Header[프로필 헤더 영역]
    BioPage --> SocialIcons[소셜 링크 아이콘 영역]
    BioPage --> LinkList[링크 카드 목록 영역]

    Header --> Avatar[프로필 아바타 이미지]
    Header --> ProfileName[닉네임]
    Header --> BioText[한 줄 소개글]

    LinkList --> LinkCard[텍스트/URL 링크 카드]

    %% 관리자 페이지 컴포넌트
    AdminPage --> SideNav[사이드 메뉴바]
    AdminPage --> MainContent[메인 편집 패널]
    AdminPage --> PreviewPanel[우측 모바일 프리뷰 패널]

    MainContent --> ProfileEdit[프로필 정보 수정 폼]
    MainContent --> LinkEditor[링크 관리자 - 추가/수정/삭제 폼]
    MainContent --> ThemeSelector[디자인 테마 선택기]

    PreviewPanel --> MockupFrame[모바일 폰 프레임 Mockup]
    MockupFrame --> MockupBioPage[프리뷰용 개인 페이지 컴포넌트]
```

## 2. ASCII 아트 와이어프레임 (Wireframe)

### 2.1. 모바일 방문자용 링크 페이지 (Mobile Bio-Link Page)
모바일 기기에 최적화된 단순하고 직관적인 레이아웃의 방문자 전용 페이지입니다.

```text
+-----------------------------------+
|               [ ]                 | <-- 프로필 아바타 이미지 (둥글게)
|                                   |
|            @creator               | <-- 크리에이터 닉네임
|    "세상의 모든 링크를 한곳에"    | <-- 한 줄 소개글
|                                   |
|       (Insta) (Youtube) (X)       | <-- 소셜 링크 아이콘 (O-03)
|                                   |
|  +-----------------------------+  |
|  |       📚 나의 포트폴리오     |  | <-- 텍스트/URL 링크 블록 1 (F-03)
|  +-----------------------------+  |
|  |      🎥 최신 유튜브 동영상   |  | <-- 텍스트/URL 링크 블록 2 (F-03)
|  +-----------------------------+  |
|  |      🛍️ 공식 굿즈 샵          |  | <-- 텍스트/URL 링크 블록 3 (F-03)
|  +-----------------------------+  |
|                                   |
|            [ MyLink ]             | <-- 서비스 로고 / 워터마크
+-----------------------------------+
```

### 2.2. PC용 관리자 대시보드 (Admin Dashboard)
좌측의 편집 영역과 우측의 실시간 모바일 프리뷰 영역으로 구성된 PC용 2단 분할 레이아웃 화면입니다.

```text
+-----------------------------------------------------------------------------------+
|  [MyLink Admin]                                                      Log out |
+-------------+----------------------------------------------+----------------------+
|             |  [ 링크 관리 ]      [ 프로필 설정 ]   [ 디자인 ] |  [ 실시간 미리보기 ]  |
|   ( Menu )  +----------------------------------------------+                      |
|             |                                              |  +----------------+  |
|  - 링크     |  +----------------------------------------+  |  |      [ ]       |  |
|    콘텐츠   |  |  + [새 링크 추가]                      |  |  |                |  |
|             |  +----------------------------------------+  |  |    @creator    |  |
|  - 디자인   |  |  [=] 📚 나의 포트폴리오          [Edit] |  |  | "세상의 모든..."|  |
|             |  |      URL: https://portfolio.com        |  |  |                |  |
|  - 애널리   |  |  [=] 🎥 최신 유튜브 동영상       [Edit] |  |  | (Insta) (Yt)   |  |
|    틱스     |  |      URL: https://youtube.com/...      |  |  | +------------+ |  |
|             |  |  [=] 🛍️ 공식 굿즈 샵             [Edit] |  |  | | 포트폴리오 | |  |
|             |  |      URL: https://shop.com             |  |  | +------------+ |  |
|             |  +----------------------------------------+  |  | | 유튜브 영상| |  |
|             |                                              |  | +------------+ |  |
|             |  [ 통계 요약 (기본) ]                          |  |  [ MyLink ]    |  |
|             |  - 총 링크 클릭 수: 1,240회                    |  +----------------+  |
|             |                                              |   (모바일 프리뷰 화면) |
+-------------+----------------------------------------------+----------------------+
```
