# 마이링크 (MyLink) 제품 요구사항 정의서 (PRD)

## 1. 프로젝트 개요
- **프로젝트명**: 마이링크 (MyLink)
- **목적**: 인스타그램, 틱톡, 유튜브 등 다양한 SNS 플랫폼에서 활동하는 크리에이터와 인플루언서들이 자신만의 단일 링크(바이오 링크) 페이지를 통해 모든 소셜 미디어 채널, 포트폴리오, 외부 콘텐츠를 한곳에 모아 보여줄 수 있는 링크트리(Linktree) 클론 서비스.
- **대상 사용자**:
  - 다중 SNS 채널을 운영하는 크리에이터 및 인플루언서
  - 단일 링크로 포트폴리오나 콘텐츠를 소개하고 싶은 개인 사용자
  - 자신을 브랜딩하고자 하는 1인 지식 창업자 및 프리랜서

## 3. 기능 상세 설명

### 3.1. 회원가입 및 사용자 인증
#### F-01: 회원가입 및 로그인 (필수)
- **설명**: 사용자는 이메일과 비밀번호를 입력하여 계정을 생성할 수 있다.
- **상세 사양**:
  - 비밀번호 규칙: 최소 8자 이상, 영문/숫자/특수문자 조합 권장.
  - 이메일 중복 체크 및 이메일 형식 유효성 검사 진행.
  - 패스워드 분실 시 재설정 기능 제공 (비밀번호 변경 링크를 이메일로 발송).
#### O-01: 소셜 로그인 (선택)
- **설명**: 구글, 카카오 등 주요 소셜 계정을 사용하여 간편하게 회원가입 및 로그인을 수행할 수 있다.
- **상세 사양**:
  - 최초 소셜 로그인 시, 자동으로 회원가입 절차 진행 및 기본 프로필 설정.
  - 기존 이메일 가입자와 소셜 계정 이메일이 일치할 시 계정 연동 지원 여부 검토.

### 3.2. 프로필 및 개인 링크 페이지 생성
#### F-02: 개인 링크 페이지 생성 (필수)
- **설명**: 가입 시 혹은 최초 설정 시 고유한 사용자 ID(username)를 설정하여 개인용 바이오 링크 URL을 부여받는다.
- **상세 사양**:
  - 주소 형식: `https://mylink.me/[username]`
  - username은 영문 소문자, 숫자, 하이픈(-) 및 언더바(_)만 허용하며 중복될 수 없음.
  - 한 번 설정한 username은 원칙적으로 수정 불가능하나, 설정 페이지에서 제한된 횟수로 수정 가능하게 지원.
#### F-04: 프로필 설정 (필수)
- **설명**: 개인 페이지 상단에 노출되는 프로필 정보를 설정한다.
- **상세 사양**:
  - 프로필 이미지 업로드: 최대 5MB, JPG/PNG 지원. 기본 프로필 아바타 제공.
  - 프로필 닉네임 설정 (최대 20자).
  - 한 줄 소개 및 자기소개글 작성 (최대 100자).

### 3.3. 링크 및 컴포넌트 관리 (콘텐츠 블록)
#### F-03: 텍스트/URL 링크 블록 (필수)
- **설명**: 사용자는 자신의 페이지에 노출할 외부 링크를 카드 형태로 생성하고 관리한다.
- **상세 사양**:
  - 새 링크 추가 버튼을 통해 타이틀(최대 50자)과 목적지 URL을 입력하여 저장.
  - 링크의 활성화/비활성화 토글 기능 제공 (비활성화 시 페이지에서 숨김 처리되지만 데이터는 유지).
  - 드래그 앤 드롭(Drag and Drop) 방식을 활용한 링크 블록 간의 노출 순서 실시간 재배치 지원.
  - 개별 링크 블록 삭제 기능.
#### O-02: 확장 미디어 블록 (선택)
- **설명**: 단순 텍스트 링크 외에 다양한 포맷의 미디어 콘텐츠를 페이지에 노출한다.
- **상세 사양**:
  - **동영상 임embed**: 유튜브(YouTube) 또는 틱톡(TikTok) 동영상 URL을 입력하면, 페이지 내에서 플레이어가 직접 렌더링되어 재생 가능.
  - **이미지 갤러리**: 이미지 파일 업로드 및 링크 연결 기능. 바둑판식(Grid) 혹은 슬라이드쇼(Carousel) 형태로 노출.
#### O-03: SNS 소셜 아이콘 연동 (선택)
- **설명**: 사용자의 주요 소셜 미디어 채널 링크를 깔끔한 아이콘 모음으로 정렬하여 보여준다.
- **상세 사양**:
  - 인스타그램, 유튜브, 페이스북, 트위터(X), 틱톡, 깃허브, 이메일 등의 URL 입력 필드 제공.
  - 입력된 링크에 맞게 브랜드 공식 로고 아이콘이 페이지 하단 혹은 프로필 영역 하단에 정렬되어 표시됨.

### 3.4. 디자인 및 브랜딩
#### F-05: 기본 디자인 관리 (필수)
- **설명**: 사전 정의된 테마 중 하나를 선택하여 개인 페이지의 룩앤필(Look & Feel)을 빠르게 변경한다.
- **상세 사양**:
  - 최소 5종 이상의 기본 테마 제공 (라이트 테마, 다크 테마, 파스텔 테마, 미니멀 테마, 비비드 테마).
  - 테마 변경 시 배경색, 버튼 모양 및 둥글기, 텍스트 색상이 테마 프리셋에 맞춰 일괄 변경됨.
#### O-05: 사용자 정의 디자인 (선택)
- **설명**: 정형화된 프리셋 외에 세부적인 디자인 요소를 직접 커스터마이징한다.
- **상세 사양**:
  - **배경 설정**: 단색(Color Picker), 그라데이션 조합 설정 또는 배경 이미지 직접 업로드.
  - **버튼 스타일**: 버튼 채우기(Solid) / 테두리만 노출(Outline) / 그림자 효과(Shadow) 스타일, 모서리 곡률(Roundness) 조정.
  - **폰트 설정**: 서비스 제공 기본 폰트(Noto Sans, Inter, Outfit 등) 중 페이지 전체 폰트 선택.

### 3.5. 통계 및 분석 (Analytics)
#### F-06: 링크 클릭 통계 (기본) (필수)
- **설명**: 관리자 페이지 내에서 각 링크 블록이 클릭된 누적 횟수를 확인한다.
- **상세 사양**:
  - 관리자 대시보드 화면에 현재 등록된 모든 링크 목록과 링크별 누적 클릭 카운트 노출.
#### O-04: 상세 분석 대시보드 (선택)
- **설명**: 단순 누적 클릭 외에 유의미한 분석 데이터를 시각화하여 제공한다.
- **상세 사양**:
  - 기간 필터(최근 7일, 최근 30일, 사용자 정의 기간) 제공.
  - 일별/주별 방문자 수(Page Views, Unique Visitors) 추이 그래프.
  - 유입 경로(Referrer) 분석 (예: 인스타그램, 틱톡, 네이버, 직접 유입 등).

### 3.6. SEO 및 시스템 최적화
#### O-06: 사용자 정의 SEO 설정 (선택)
- **설명**: 개인 링크 페이지가 포털 사이트에 검색되거나 SNS에 공유될 때의 정보를 직접 수정한다.
- **상세 사양**:
  - 메타 타이틀(Title) 및 메타 설명(Description) 직접 지정 기능.
  - SNS(카카오톡, 페이스북, 슬랙 등) 공유 시 표시될 대표 이미지(Open Graph Image) 업로드 기능.

## 4. 비기능적 요구사항
- **반응형 웹 디자인 (Mobile-First)**: 모바일 브라우저 환경에서 최적화된 사용자 경험(UX)을 보장해야 하며, 태블릿 및 데스크톱 브라우저에서도 어색하지 않은 와이드/미디엄 뷰포트 레이아웃 제공.
- **성능 및 속도**: 핵심 페이지(사용자 개인 링크 페이지) 로딩 속도 최적화 (첫 바이트 시간 200ms 이하, 완전 로딩 1.5초 이하). 이미지 최적화(WebP 포맷 변환 및 Lazy Loading) 탑재.
- **보안**: HTTPS 프로토콜 강제 적용, 사용자 비밀번호의 안전한 해싱(BCrypt 등 사용). Cross-Site Scripting(XSS) 및 CSRF 방지 처리.
- **접근성**: 스크린 리더 사용자도 탐색할 수 있도록 시맨틱 마크업을 준수하고 필수 `aria-*` 속성 사용.
