# 마이링크 (MyLink) 사용자 시나리오

## 1. 방문자 시나리오 (Visitor Scenario)
방문자가 크리에이터의 마이링크 페이지에 방문하여 원하는 정보로 이동하는 흐름입니다.

- **페이지 접속**
  - 사용자는 크리에이터가 공유한 전체 링크 목록을 확인하기 위해 외부 SNS(인스타그램, 유튜브 등) 프로필에 등록된 마이링크 고유 URL(`https://mylink.me/username`)을 터치한다.
- **프로필 정보 확인**
  - 사용자는 방문한 페이지가 자신이 찾는 크리에이터의 공식 페이지인지 식별하기 위해 페이지 상단에 표시된 프로필 이미지, 닉네임, 그리고 소개글을 확인한다.
- **링크 탐색 및 클릭**
  - 사용자는 크리에이터가 추천하는 외부 콘텐츠나 상품 정보로 이동하기 위해 목록에 나열된 텍스트/URL 링크 카드 중 원하는 항목을 터치한다.
  - 사용자는 원래 탐색하던 페이지를 유지하면서 외부 사이트로 이동하기 위해 새 창(혹은 새 탭)으로 열리는 목적지 페이지로 접속한다.

---

## 2. 소유자 시나리오 (Owner Scenario)
크리에이터(소유자)가 서비스에 가입하고 본인의 프로필 및 링크를 관리하는 흐름입니다.

- **회원가입 및 계정 생성**
  - 사용자는 자신만의 마이링크 페이지를 개설하기 위해 회원가입 화면에서 사용 중인 이메일 주소와 비밀번호를 입력하여 계정을 생성한다.
  - 사용자는 서비스 내에서 사용할 고유한 개인 페이지 주소를 선점하기 위해 중복되지 않는 고유 사용자 ID(username)를 설정한다.
- **프로필 초기 설정**
  - 사용자는 방문자들에게 본인의 브랜드 이미지를 효과적으로 전달하기 위해 설정 메뉴에서 대표 프로필 이미지를 업로드한다.
  - 사용자는 본인의 정체성을 설명하기 위해 노출될 프로필 닉네임을 입력하고 한 줄 소개글을 작성하여 저장한다.
- **링크 추가**
  - 사용자는 방문자들에게 소개하고 싶은 새로운 외부 링크를 등록하기 위해 관리 화면에서 '새 링크 추가' 버튼을 선택한다.
  - 사용자는 방문자가 어떤 링크인지 명확히 인지하게 돕기 위해 노출할 링크 제목(타이틀)과 이동할 URL 주소를 입력하여 링크 카드를 생성한다.
- **링크 수정**
  - 사용자는 기존에 등록해 둔 링크의 설명이나 이동 주소를 최신 정보로 갱신하기 위해 해당 링크 카드의 수정 버튼을 누른다.
  - 사용자는 변경된 타이틀이나 URL 주소를 다시 입력하고 저장 버튼을 눌러 개인 페이지에 즉시 반영되도록 한다.
- **링크 삭제**
  - 사용자는 프로필 페이지에 더 이상 노출할 필요가 없는 만료된 링크를 정리하기 위해 해당 링크 카드의 삭제 버튼을 선택한다.
  - 사용자는 실수로 링크가 지워지는 현상을 방지하기 위해 노출되는 최종 삭제 확인창에서 삭제 완료를 선택하여 페이지에서 링크를 지운다.
