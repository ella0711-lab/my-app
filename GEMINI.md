# Project Instructions

This is a Next.js application using the App Router, TypeScript, and TailwindCSS.

## Project Overview

- **Framework**: Next.js (App Router)
- **Language**: TypeScript
- **Styling**: TailwindCSS
- **Linting**: ESLint

## Building and Running

The following scripts are available in `package.json`:

- `npm run dev`: Starts the development server.
- `npm run build`: Builds the application for production.
- `npm run start`: Starts the production server (after building).
- `npm run lint`: Runs ESLint to check for code quality.

## Development Conventions

- **File Structure**: Use the standard Next.js `app/` directory structure for routes and components.
- **Styling**: Utilize TailwindCSS for styling.
- **Types**: Ensure all new files are typed with TypeScript.
- **Linting**: Run `npm run lint` before committing changes to ensure code quality and adherence to project standards.
